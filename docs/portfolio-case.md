# Portfolio case — Metric Contract Studio

## Case framing

Many analytics portfolios show dashboards. This project shows **governance**: how a data professional prevents metric chaos before visualization.

## Narrative arc

1. **Problem:** conflicting KPI definitions across teams
2. **Product:** contract studio with validation + maturity scoring
3. **Craft:** TypeScript domain model, pure testable rules, internal-tool UX
4. **Outcome:** exportable documentation artifact (Markdown + SQL template)

## Evidence to show in interviews

- Demo metrics with realistic SaaS/e-commerce semantics
- Critical vs warning validation policy
- Explainable score breakdown
- SQL template generation by metric type
- Clean information architecture for an internal data tool
